# PY_Attano
使用純Python進行撰寫的bot，以discord.py模組編寫。

## 說明
Just a lan bot.

## 我也想要一份自己的
自己來。
可以直接fork這份專案，把`setting.json`裡面的token改成你自己的就好。
然後功能可以自己去看Cog跟最基本的一些教學，自己學。

## 注意
bot採用Python 3.8編寫
